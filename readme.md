## To Start
```
yarn install
```
Start development server:
```
yarn start
```

## Dev Server Port

You can change the dev server's port number by modifying the `vite.config.js` file. Look for the `server` section:

```js
{
	// ...
	server: { host: '0.0.0.0', port: 5173 },
}
```

Change 8000 to whatever you want.
